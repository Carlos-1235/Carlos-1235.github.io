Avocado Market Analysis

Python • Pandas • Tableau

Challenge: Understanding avocado price dynamics across regions and time can be difficult due to variability in supply, product type (organic vs conventional), and market distribution. Businesses lack clear visibility into pricing trends and the factors influencing them, limiting data-driven decision-making.

Solution: Analyzed a dataset of over 18,000 avocado sales records using Python (Pandas) to perform data cleaning and exploratory data analysis (EDA). Explored price trends over time, regional price differences, and the relationship between total volume and pricing. Developed an interactive Tableau dashboard to visualize key market dynamics. The dashboard includes a time series analysis with regional filtering, a ranked comparison of average prices across regions, a comparison between organic and conventional products, and a scatter plot to analyze the relationship between supply (volume) and price.

Impact: Identified consistent pricing patterns across regions and confirmed a price premium for organic avocados. The analysis also revealed a negative relationship between supply (volume) and price, supporting key supply-demand dynamics. The dashboard improves visibility into market behavior, enabling users to explore regional differences and price trends over time.

Telecommunications Operator Performance Analysis

Python • Pandas • Tableau

Challenge: The telecommunications company lacked visibility into operator performance, making it difficult to detect inefficiencies such as missed calls, long wait times, and uneven workload distribution.

Solution: Analyzed a dataset of over 53,000 call records using Python (Pandas) to perform exploratory data analysis (EDA). Defined performance benchmarks using percentile-based thresholds and identified inefficient operators based on missed calls, wait time, and outbound call activity. Additionally, conducted a hypothesis test (α = 0.05) to validate statistically significant differences between efficient and inefficient operators. Built interactive dashboards in Tableau to improve performance monitoring and reporting.

Impact: Identified 212 underperforming operators and validated performance gaps through statistical analysis, enabling data-driven optimization of call handling and efficiency.

A/B Testing - Recommender System

Python • Pandas • A/B Testing

Challenge: Evaluate whether a new recommender system could increase conversion rates by more than 10%, while ensuring the experiment results were reliable and free from bias.

Solution:Analyzed over 423,000 user events across the conversion funnel (product page → cart → purchase) using Python (Pandas). Cleaned and validated A/B test data, identifying critical issues such as duplicate users assigned to both groups, which reduced sample integrity. Performed exploratory data analysis (EDA) and conducted z-tests for proportions (α = 0.05) to compare conversion rates between control (Group A) and test (Group B).

Impact: Demonstrated that the new recommender system failed to improve conversions and identified critical experimental flaws, preventing a potentially harmful product rollout.
